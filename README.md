# HatchDragonsDB-js

A JavaScript port of the original **HatchDragonsDB** project, adapted to run as a static web application and deploy easily with GitHub Pages.

Live version:
https://sleepydarian.github.io/HatchDragonsDB-js/

## Origin

This project is forked from:

https://github.com/zechary28/HatchDragonsDB

The original project was adapted and ported to JavaScript so it could be deployed as a static site using GitHub Pages.

## About

HatchDragonsDB-js is a searchable database for dragons from the game **Hatch Dragons**.

The project provides a simple interface to browse and search dragon information, including:

- Dragon name
- Dragon set
- Rarity
- Essence
- Type

It is designed as a lightweight frontend-only application that can run directly in the browser.

## Technology

Built with:

- HTML
- CSS
- JavaScript
- SQLite (via sql.js / WebAssembly)

The database is loaded locally in the browser, so no backend server is required.
